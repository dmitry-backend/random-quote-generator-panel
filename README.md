# Random Quote Generator Panel
A basic frontend JavaScript project demonstrating clean, professional coding practices and structured logic.

## Overview
This project allows users to click a button to see a new random motivational quote.
The same quote can’t follow twice in a row.
No default submit behavior for the button.

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)

## Project Structure
```text
random-quote-generator-panel/
├── index.html
├── styles.css
├── script.js
├── README.md
└── LICENSE
```

## Installation
1. Clone the repository:
```bash
git clone https://github.com/dmitry-backend/random-quote-generator-panel
```
2. Navigate into the project directory:
```bash
cd random-quote-generator-panel
```

## Running the Application
Open the `index.html` file in your preferred browser. No server setup is required.
```bash
open index.html
```
or double-click the file in your file explorer.
